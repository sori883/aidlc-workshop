## green

```
Using AI-DLC, 社内向けの会議室予約システムを作りたいです。

背景: 現在は共有スプレッドシートで会議室を管理しており、
ダブルブッキングが頻発して困っています。

必須と考えていること:
- 会議室の登録・予約・キャンセルができること
- 予約の重複を防げること

制約:
- Python + FastAPI + SQLite、ローカル実行で完結
- フロントエンドは不要（REST APIのみ）
- ワークショップなので2〜3時間で完成する範囲に収めてください

細かい仕様（定期予約、権限、キャンセルポリシーなど）は
決めていないので、質問してください。
```
